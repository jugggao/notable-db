# Shell 内建命令 - 变量处理 - `eval`

## 用法

### 语法

```shell
eval [arg ...]
```

### 作用

`eval` 获取其参数，将它们用空格连接起来，其中包含的任何变量或表达式都会被展开，然后再当前执行环境中将生成的字符串作为 Bash 代码执行。对于从命令行或脚本中生成代码很有用。

`evel` 在 Bash 中的工作方式与大多数其他具有 `eval` 功能的语言基本相同。
