# Shell 内建命令 - 变量处理 - `set`

## 用法

### 语法

```shell
set [-abefhkmnptuvxBCHP] [-o option-name] [--] [arg ...]
```

### 作用

`set` 主要作用有两个：

- 将位置参数设置为 `arg ...`
- 通过选项（短选项和长选项）设置 Shell 属性

### 选项

**短选项**：

```
-a      
```
