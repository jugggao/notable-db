# Shell - 特殊字符

## `#` - 注释符

注释符。如果一行脚本开头是 `#`（除了 `#!`），那么代表这一行是注释，不会被执行。

```shell
# 这是一行注释
```

注释也可能会在一行命令结束之后出现。

```shell
echo "A comment will follow" # 这可以写注释，注意 # 之前有空格
```

注释也可以在一行开头的空白符（whitespace）之后。

```shell
    # 这个注释前面有一个制表符（tab）
```
