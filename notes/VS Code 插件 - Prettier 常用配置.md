# VS Code 插件 - Prettier 常用配置

```json
{
	"prettier.printWidth": 100, // 超过最大值换行
    "prettier.tabWidth": 4, // 缩进字节数
    "prettier.useTabs": false, // 缩进不使用 Tab，使用空格
    "prettier.singleQuote": true, // 使用单引号代替双引号
}
```
