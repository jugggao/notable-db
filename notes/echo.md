# `echo`

## 作用

打印一个字符串或变量到标准输出 `stdout`。

## 语法

```shell
echo [选项]... [字符串或变量]...
```

## 选项

```
  -n	结尾不换行
  -e	启用转义字符
  -E	不启用转义字符（默认）

使用 `-e` 选项时，若字符串出现以下字符，则进行特殊处理：

  \\      \ 字符
  \a      发出警告声音（BEL）
  \b      删除前一个字符
  \c      直接结尾并不换行（后面的字符串不会输出）
  \e      删除后一个字符
  \f      换页
  \n      换行
  \r      回车符
  \t      水平制表符
  \v      垂直制表符
```
