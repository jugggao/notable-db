# Shell 内建命令 - 变量处理 - `let`

## 作用

对变量执行算数运算。大多数情况下，它作为 `expr` 的简单版本。

## 用法

### 语法

```shell
let arg [arg ...]
```

### 运算符

优先级递减。

```
id++, id--			变量赋值后递增，递减
++id, --id			变量赋值前递增，递减
-, +				负，正
!, ~				逻辑否，按位取反
**					幂运算
*, /, %				乘，除，取余
+, -				加，减
<<, >>				按位左移，按位右移
<=, >=, <, >		比较
==, !=				等于，不等于
&					按位与
^					按位异或
|					按位或
&&					逻辑与
||					逻辑或
expr ? expr : expr	条件运算符（三元运算符）
=, *=, /=, %=,
+=, -=, <<=, >>=,
&=, ^=, |=          赋值
```
